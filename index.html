My Work

The Enviro Dashboard

This website tries to raise awareness about impact of climate change on our world. By entering the name of the city, the user will be presented with relevant climate information in the form of pollution widget. The user can take a quiz on various topics related to climate change. The website make API call to the Guardian website to display articles related to climate change.

Weather Dashboard

This is a weather dashboard which can be used to look up weather for a city. For a given city, current weather and 5-day forecast can be displayed. It also shows previously searched cities for a quick search. One key feature is color coded U/V index which shows severity of the U/V index for a city.

Online Health Portal

This is an online portal that connects patients with the doctors. In today's time of covid-19, it has become imperative to have digital channels for the delivery of healthcare. The patients can use this portal to add, edit or cancel their appointments with the doctor. The doctors can manage their patients and appointments

The Tech Blog

This app is about tech blog where users can add or edit their blogs. Users have the ability to register and log in to the blog website. Once logged in, they have the ability to create new blogs, edit existing blogs and add comments on blogs created by other users. It uses express.js, mysql, handlebars and javaSCript/html/css.
